# 🃏 Boršická Odrbávačka – Poker Timer

Poker tournament timer s plnou správou struktury turnaje.

## Lokální spuštění

```bash
npm install
npm run dev
```

## Deploy na GitHub Pages

1. Nahraj tento repozitář na GitHub
2. Jdi do **Settings → Pages**
3. Pod *Source* vyber **GitHub Actions**
4. Pushni na `main` — deploy proběhne automaticky

App bude dostupná na `https://<tvoje-jmeno>.github.io/<nazev-repa>/`
